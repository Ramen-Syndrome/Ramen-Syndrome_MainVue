## 무인 컵라면 매장 영업 관리 시스템
### 요구사항 및 담당
- 상품별 매출내역 - 김민성
- 지점별 매출 현황 - 우지영
- 과거 판매 내역 - 정수민
- 일자별 매출 현황 - 박민희

- ### 제약조건
1. 모바일에서는 사용불가. 오직 윈도우 웹에서만 작동 가능하다.
2. 대시보드만 만드는 프로젝트이다. (클릭 금지!)
3. 가상의 컵라면 체인점을 대상으로 한 영업관리 시스템입니다.
4. 2024년부터 영업을 시작했습니다
